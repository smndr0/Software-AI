Software AI Landing Page

A premium, modern landing page built to showcase the "Software AI" platform concept. This project focuses on high-performance web development, modern UI/UX principles, and seamless interactions without relying on heavy frameworks.

Overview

This repository contains a fully responsive, pixel-perfect landing page designed with a sleek, dark-themed, glassmorphic aesthetic. The goal of this project is to demonstrate advanced frontend development skills, including complex CSS animations, responsive grid layouts, and clean, semantic HTML structure.

Key Features

- Modern UI/UX: Implements a cutting-edge dark mode design with glassmorphism, subtle glowing gradients, and high-quality typography.
- Performance Optimized: Built without heavy JavaScript frameworks. Pure HTML, CSS, and Vanilla JavaScript ensure maximum performance, zero bloat, and minimal load times.
- Responsive Engineering: Fluid design that adapts perfectly to desktop, tablet, and mobile viewing environments using modern CSS Grid and Flexbox techniques.
- Complex Animations: Features custom CSS keyframe animations, hover physics, floating 3D dashboard cards, and an infinite looping marquee.
- Interactive Elements: Includes a custom animated hero section, interactive pricing toggles, and dynamic accordion features.

Tech Stack

- HTML5: Semantic tags for improved accessibility and SEO.
- CSS3: Native CSS variables for design token management, CSS Grid/Flexbox for layout, and advanced keyframe animations.
- Vanilla JavaScript: Lightweight interactions and dynamic UI state management.
- Typography: Integrated with Google Fonts (`Inter` and `Space Grotesk`) for a premium look and feel.

Getting Started

There are no dependencies or build steps required. To run this project locally:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd <repository-directory>
   ```
3. Open `index.html` in any modern web browser, or use a local development server (like VS Code Live Server) for a better developer experience.

Project Structure

- `index.html` — The main markup file containing the scalable, semantic document structure.
- `style.css` — The complete design system, including CSS variables, utility classes, component styling, and responsive media queries.
- `main.js` — Handles client-side interactivity, such as the mobile navigation menu, pricing toggles, and scroll effects.
